# Runi Empire Barber Shop Booking System

## Description
Runi Empire is a barber shop booking system developed as a progressive web application using PHP and MySQL. The system allows users to view services and eventually book appointments, while an administrative interface supports basic management of users, bookings, and service data. The project is built incrementally across weekly development cycles, focusing on structured version control, database integration, and full-stack web development practices using a local XAMPP environment.

## Tech Stack
- PHP
- MySQL
- Apache (XAMPP)
- HTML
- CSS
- JavaScript

## Setup Instructions
1. Install XAMPP
2. Clone repository into `htdocs`
3. Start Apache and MySQL from XAMPP control panel
4. Open browser and navigate to:
   http://localhost/RuniEmpire/
5. Import database files from each week folder when required
